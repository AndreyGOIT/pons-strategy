# PONS – Näyttö Strategy 2026

## Overview

This repository contains the strategic documentation and competence demonstration materials for the Pons.fi project.

Pons.fi is a real-world full-stack web application developed for a local sports organization.  
This repository focuses not on implementation, but on demonstrating professional software development competence in the context of a näyttö (final evaluation).

---

## Why This Project Matters

This project is based on a real operational need in a local sports organization.

Before the system:

- Enrollment was handled manually via email
- Information was fragmented
- Administrative workload was high

After introducing Pons.fi:

- Processes became structured and transparent
- Data handling improved
- Administrative efficiency increased

This project demonstrates how software development can directly improve real-world workflows.

## Project Goal

The purpose of this repository is to transform an existing working application into a structured, professionally documented project that meets high-level evaluation criteria.

This includes demonstrating:

- Architectural understanding
- Structured planning and prioritization
- Professional Git workflow
- Deployment and DevOps awareness
- Security considerations
- Ability to evaluate and improve own work
